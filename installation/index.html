Music Web Player - Instalation Tutorial

1° Step

Add the following code on head of your page

                                        
    <link href="player/css/style.css" rel="stylesheet"/>
    <link href="player/css/landscape.css" rel="stylesheet" media="(max-height: 400px) and (max-width: 700px)">
    <link href="player/css/animations.css" rel="stylesheet"/>
<script src="player/js/player.js"></script>

2° Step

Add the following code on end of your body page

3° Step

Add the following code on head of your page

                                        
    <!-- Player -->
    <div id="player-page" class="player-page">
        <a id="icon-player-close" href="javascript:void(0)" onclick="closePlayer()">
            &times;
        </a>

        <div class="player-page-content">
            <div class="player-box-rounded" id="player">
                <div class="player-content" id="player-content">

                    <div class="player-cover">
                        <img id="music-cover" src="player/js/img/BMTH.jpg">
                    </div>

                </div>
                <div class="player-controller" id="player-controller">
                    <audio id="audio">
                        <source src="player/js/musics/Follow_You.mp3" type="audio/mpeg">
                        Your browser does not support our player
                    </audio>

                    <div class="time" id="time">
                        <p id="actual-time">00:00:00</p>
                        <progress id="progress-bar" max="0" value="0"></progress>
                        <p id="total-time">00:00:00</p>
                    </div>

                    <div class="player-buttons">
                        <a class="control-button button-prev" onclick="prev()">
                            <i class="glyphicon glyphicon-backward" aria-hidden="true"></i>
                        </a>

                        <a class="control-button button-play" onclick="play()" id="button-play">
                            <span class="glyphicon glyphicon-play"></span>
                        </a>
                        
                        <a class="control-button button-pause" onclick="pause()" id="button-pause">
                            <i class="glyphicon glyphicon-pause" aria-hidden="true"></i>
                        </a>

                        <a class="control-button button-next" onclick="next()">
                            <i class="glyphicon glyphicon-forward" aria-hidden="true"></i>
                        </a>

                        <a class="control-button button-volume">
                            <i class="glyphicon glyphicon-volume-up" id="volume-icon" aria-hidden="true"></i>
                            <div class="dropdown-content">
                                <input type="range" min="0" id="volume" max="100" />
                            </div>
                        </a>
                    </div>
                </div>
            </div>
        </div>
    </div>

4° Step

And finally, you just have to add this button wherever you want

                                        
    <a id="icon-player" href="javascript:void(0)" onclick="openPlayer()">Try It</a>
